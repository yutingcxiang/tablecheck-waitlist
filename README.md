## Getting Started
To build the app locally, use:
```
docker-compose build
```

To run the app locally in the background, use:
```
docker-compose up -d
```

To stop the app, use:
```
docker-compose stop
```

To access the app, navigate to `http://localhost:3002/` for the api and `http://localhost:3000/` for the client.
